# JavaScript Programming Guide

## Variables and Data Types

JavaScript has several data types:

- String
- Number
- Boolean
- Object
- Array
- Null
- Undefined

## Functions

Functions are first-class citizens in JavaScript. They can be:

- Assigned to variables
- Passed as arguments
- Returned from other functions

Example:

```JavaScript
function greet(name) {
    return `Hello, ${name}!`;
}
```
